# Twitter Sentiment Analysis Dashboard (AI, ML & Data Science)

## Project Overview

This project analyzes **public sentiment on Artificial Intelligence, Machine Learning, and Data Science** using real-time Twitter data. The aim is to understand how people perceive these technologies and present the insights through an **interactive Power BI dashboard** for easy interpretation and decision-making.

## Domain

**Technology Analytics / Social Media Analytics**

## Objectives

* Collect real-time tweets related to AI, ML, and Data Science
* Clean and preprocess unstructured tweet text
* Perform sentiment analysis to classify opinions as Positive, Negative, or Neutral
* Visualize sentiment trends and distributions using Power BI

## Tech Stack

* **Data Source:** Twitter API v2
* **Libraries:** Tweepy, Pandas, Regex, VADER Sentiment Analyzer
* **Visualization Tool:** Power BI

## Outcome

The project provides **real-time insights into public opinion on AI-related topics** and demonstrates practical skills in **API integration, NLP preprocessing, sentiment analysis, and business-focused data visualization**.
